# Glow Log
빛나는 변화를 만드는 습관기록 앱

작은 습관이 모여, 당신을 빛나게 합니다
Glow Log는 달에서 해까지 이어지는 빛의 여정을 담은 습관 기록 앱입니다

매일의 작은 실천을 기록하며, 눈에 보이는 성취감과 변화를 경험해보세요
